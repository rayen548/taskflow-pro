# TaskFlow Pro - Frontend

A beautiful, modern React application built with Vite.

## Features

- Task management interface
- User management interface
- Real-time statistics dashboard
- Responsive design
- Modern UI with gradients and animations
- Full API integration with FastAPI backend

## Technologies

- React 18
- Vite
- Axios for API calls
- React Icons
- Modern CSS with animations

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The app will be available at http://localhost:3000

## Build for Production

```bash
npm run build
```

## API Configuration

The frontend connects to the FastAPI backend at `http://localhost:8000`. Make sure the backend is running before starting the frontend.
